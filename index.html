Topic-Based Content Management System (TBCMS) using C++.

IoT Based Gas Leakage Detector

As part of my hands-on learning experience, I developed a smart alert system using NodeMCU ESP8266, gas sensors, servo motor, buzzer, and LED indicators, all connected via a breadboard. The system is integrated with the Blynk IoT platform to send real-time notifications via email and the mobile app in the event of gas leakage detection. This project showcases my ability to work with microcontrollers, IoT communication, and real-time monitoring, demonstrating both my technical skills and my commitment to building practical, safety-focused solutions.
